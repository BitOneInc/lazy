## Usage

### Learn something..
```
/learn hi - greeting
```


### Add some greeting message..
```
/add greeting - Hello there!
/add greeting - Hello buddy!
```

### Show categories
```
/categories
```
### Show responses
```
/responses greeting
```
### Just quiet
```
/quiet
```
